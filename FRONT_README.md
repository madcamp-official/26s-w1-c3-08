# 프론트엔드 변경 기록

## 2026-07-07

- `frontend` 브랜치에서 작업했고, 이번 변경은 `apps/web`의 화면 표현 레이어 안에서만 진행했다.
- 메인 화면의 `곧 찾아갈 마음` 패널에서 고정 높이를 제거해, 호출된 텍스트 길이에 맞춰 흰색 상자가 자연스럽게 늘어나도록 조정했다.
- `마음 보내기` 화면의 전달 설정 패널에서 전달 예정 시간 입력 영역과 도착 예정 요약 문구가 박스 밖으로 나가지 않도록 폭, 그리드, 줄바꿈을 보정했다.
- `마음 보내기` 화면의 `익명으로 보내기`, `도착 예정일 숨기기` 스위치를 버튼형 토글로 바꿔, 클릭 시 왼쪽 `오늘의 한 줄` 배경이 함께 보라색으로 변하지 않도록 수정했다.
- `친구` 화면의 패널과 목록 행 폭을 정리해 상자들이 같은 가로 기준으로 정렬되도록 맞췄다.
- `친구`와 `내 정보` 화면의 상자형 히어로 헤더를 제거하고, `마음 보내기` 화면처럼 제목과 소개 문구가 배경 위에 바로 나오도록 통일했다.
- `내 정보` 화면에서 별도 `기타` 섹션을 제거하고, 개인정보 보호 안내 상자 우측 하단에 로그아웃 버튼을 배치했다.
- `maeari_logo.png`를 웹 정적 이미지로 추가하고, 상단 바와 공개 화면에서 사용하던 매아리 로고 이미지를 새 파일로 교체했다.
- 적용했던 KoPubWorld 바탕과 송명 폰트 설정을 제거해 기존 기본 글꼴 체계로 되돌렸다.
- 친구 코드 복사 버튼 간격과 친구 요청 폼의 버튼 넘침은 유지한 채, 친구 코드 글꼴은 기존 고정폭 스타일로 되돌렸다.
- 메인 화면의 `최근 보관한 마음` 섹션을 전체 콘텐츠 폭에 맞추고, 보관된 마음이 없을 때 프론트엔드 빈 상태 안내 상자를 보여주도록 수정했다.
- 페이지 제목과 메인 화면의 주요 제목 문구에 충주김생체를 적용하고, 현재 KST 시간 값은 KoPubWorld 바탕 Light로 표시하도록 조정했다.
- `현재 KST`, `현재 시각 (KST)` 라벨도 KoPubWorld 바탕 Light로 통일했다.
- 로고의 `매아리` 글씨는 학교안심 우주체로 바꾸고, 충주김생체를 적용했던 제목 계열 글씨는 부크크 명조 700으로 교체했다.
- 제목 계열 글씨를 부크크 명조 700에서 마포애민체로 다시 교체했다.
- 제목 계열 글씨를 KoPubWorld 바탕 Bold로 교체하고, 자간을 더 조밀하게 조정했다.
- 메인 히어로를 `maeari-hero-night.png` 배경 이미지 기반의 반응형 배너로 바꾸고, `곧 찾아갈 마음` 영역을 전체 폭 가로형 패널로 정리했다.
- 메인 히어로 배경 이미지의 기본 위치를 아래쪽 기준으로 맞춰 위쪽이 먼저 잘리도록 하고, 배너 글씨색을 이미지 톤에 맞게 조정했다.
- 친구 화면의 주요 섹션 제목을 볼드로 통일하고, 내 친구 코드 복사 버튼을 요청/찾기 버튼과 같은 primary 색 조합으로 맞췄다.
- 메인 히어로 라벨 글씨색을 기존 연보라 계열로 되돌리고, 홈 하단 바로가기 카드 섹션을 제거했다.
- 사이드바 메뉴가 좁은 세로 화면에서도 우선 보이도록 `오늘의 한 줄` 배치를 조정하고, 해당 문구를 KoPubWorld 바탕 제목/본문 굵기로 나눠 적용했다.
- 좌측 사이드바의 `오늘의 한 줄` 숨김 조건을 제거하고, 메뉴 아래에 보이도록 더 작은 카드 형태로 압축했다.
- 좌측 사이드바 주요 메뉴에서 `내 정보` 항목을 제거했다.
- 큰 화면에서 홈 히어로 우측에 `곧 찾아갈 마음` 타임라인이 다시 배치되도록 반응형 그리드를 조정했다.
- 좌측 사이드바의 `오늘의 한 줄` 카드가 고정 높이가 아니라 메뉴 아래 남은 높이를 유동적으로 차지하도록 변경했다.
- 홈 콘텐츠 최대 폭을 넓혀 큰 화면의 우측 빈 공간을 줄이고, 작은 화면에서는 `곧 찾아갈 마음`이 `최근 보관한 마음` 위에 오도록 순서를 조정했다.
- 예정된 마음이 없을 때 `곧 찾아갈 마음`에는 `마음나무` 한 항목만 보이도록 변경했다.
- 홈 히어로의 `받은 마음 보기` 아이콘을 메뉴와 같은 하트 아이콘으로 바꿨다.
- `최근 보관한 마음` 제목을 `곧 찾아갈 마음`과 같은 제목 글꼴로 맞추고, `전체 보기` 버튼을 우측 콘텐츠 라인에 맞췄다.
- 좌측 사이드바의 `오늘의 한 줄` 아이콘을 제거하고 제목을 `오늘의 한 줄🌙`로 바꿨으며, 본문 글자 크기가 카드 높이에 맞춰 줄어들 수 있도록 조정했다.
- `마음 보내기` 전달 시간 선택 버튼의 과한 볼드 굵기를 제거하고, `고마움`과 `응원` 감정 이모티콘을 교체했다.
- 열어둔 마음나무가 없을 때 화분 느낌의 아이콘과 `마음나무를 심어주세요.` 안내가 보이도록 빈 상태를 수정했다.
- 마음나무 빈 상태 문구를 `심어둔 마음나무` 표현으로 바꾸고, 메뉴의 마음나무 아이콘도 새싹 아이콘으로 통일했다.
- 감정 이모티콘에서 `응원`은 클로버로 되돌리고 `그리움`은 물방울 이모티콘으로 변경했다.
- 내 정보 화면의 연락처 인증번호 입력을 행 내부 입력칸 대신 팝업 모달에서 진행하도록 바꿨다.
- 받은 마음 앨범형 UI 구현을 위해 프론트엔드와 백엔드가 나눠 진행할 작업 계획을 `RECEIVED_ALBUM_PLAN.md`로 정리했다.
- `보낸 마음` 화면에 임시 저장을 제외한 상단 상태 탭, 기간 필터, 전체/예약함/전달 완료/전달 실패 요약 카운터를 프론트 데이터 기반으로 추가했다.
- 메인 콘텐츠의 최대 폭 제한을 제거해 모니터 폭에 맞춰 더 넓게 배치되도록 조정했다.
- 좌측 사이드바에 `/archive`로 이동하는 `마음 보관함` 메뉴를 추가하고, 기존 아카이브 화면의 사용자 노출 이름을 `마음 보관함`으로 변경했다.
- `오늘의 한 줄`은 현재 백엔드 값이 아니라 프론트 `AppShell`에 고정된 문구임을 확인했다.
- 좌측/하단 메뉴에서 `보낸 마음` 항목을 제거했다.
- 마음나무 상세의 도착 전 안내 문구를 `마음이 오는 중이에요. 도착까지 기다려 주세요.`로 변경했다.
- 메인 화면의 `최근 보관한 마음` 문구를 `최근 찾아온 마음`으로 변경했다.
- 백엔드 `0.2.5.9 마음 목록 thumbnail backend 계약`에 맞춰 메인, 받은 마음, 보낸 마음, 마음 보관함, 미래의 나 목록 카드가 `thumbnail.url`을 우선 사용하도록 반영했다. 썸네일이 없으면 기존 기본 봉투 이미지가 보이도록 fallback은 유지했다.
- 모바일 하단 메뉴에서 `보낸 마음`을 제거한 상태에 맞춰 메뉴 칸 수를 4칸으로 조정했다.
- 공개 마음나무와 받은 마음 공개/오류 화면의 상단 로고를 누르면 홈페이지로 이동하도록 링크를 연결했다.
- 메뉴에서 잘못 제거했던 `보낸 마음`을 다시 복구하고, 요청에 맞춰 `받은 마음` 항목을 좌측/하단 메뉴에서 제거했다.
- 복구한 `보낸 마음` 메뉴 아이콘을 기존과 같은 트레이 형태의 `Inbox` 아이콘으로 되돌렸다.
- `RECEIVED_ALBUM_PLAN.md` 방향에 맞춰 받은 마음과 마음 보관함 목록을 앨범형 카드 그리드로 변경했다. 카드 배경은 `thumbnail.url`을 우선 사용하고, 없으면 기존 기본 봉투 이미지로 대체되도록 했다.
- 메인 화면의 `최근 찾아온 마음` 썸네일만 `KakaoTalk_Photo_2026-07-06-21-56-35 001.png`부터 `004.png`까지의 4개 이미지 중 message id 기준 고정 랜덤으로 표시하도록 변경했다.
- 보낸 마음 화면에서 공개 링크 복사 알림과 감정 태그 필터 사이에 여백을 추가하고, 공통 감정 태그 라벨에 작성 화면과 같은 이모티콘을 포함하도록 정리했다.
- 앱 전역 `loading.tsx`를 추가해 페이지 로딩 동안 편지봉투가 펼쳐지고 연보라색 별이 떠오르는 로딩 애니메이션을 표시하도록 했다.
- 로딩 애니메이션을 예시 이미지처럼 납작한 2D 편지봉투가 열리며 보라색 안감과 연보라색 별이 보이는 형태로 재조정했다.
- 마음나무 닫기 전에 예정 도착 시간을 안내하며 실제로 닫을지 확인하는 팝업을 추가했다.
- 보낸 마음함 요약에서 예약함 아이콘은 시계, 전달 실패 아이콘은 삼각형 주의 아이콘으로 변경했다.
- 앨범 카드에서 본문 미리보기 줄을 제거하고, 받은 마음 카드의 시간은 발신일 숨김 여부와 관계없이 실제 수신 시간이 표시되도록 조정했다.
- 마음 보내기 완료 팝업이 상단바 뒤에 가려지지 않도록 z-index와 세로 배치를 조정하고, 긴 내용은 팝업 내부에서 스크롤되도록 수정했다.
- 앨범 카드에서 별표 아이콘을 제거하고, 미확인 메시지 표시 동그라미에 흰색 외곽선을 추가했다.
- 보낸 마음 목록에서 썸네일을 제거하고, 마음 보내기 완료 팝업의 `보낸 마음 보기` 버튼을 숨겼다.
- 마음나무 생성 성공 알림 아래에 여백을 추가하고, 각 마음나무 카드의 `상세정보`에서 QR과 링크를 다시 확인할 수 있도록 했다.
- 마음나무 `열기` 결과가 별도 하단 영역이 아니라 해당 마음나무 카드 안에서 펼쳐지도록 변경했다.
- 마음나무 닫기 확인 팝업의 레이어를 앱 전체보다 확실히 위로 올리고 배경 흐림과 어두운 덮개를 강화해, 프로필과 좌측 메뉴가 떠 보이지 않도록 정리했다.
- 마음 보내기 완료 팝업이 상단 바에 가려지지 않도록 레이어와 중앙 배치를 조정하고, 완료 팝업에서는 QR 공유 영역을 더 작은 크기로 표시해 일반 모니터 화면에서 스크롤 없이 보이도록 개선했다.
- 마음 보내기 전달 설정 영역과 완료 팝업에서 `보낸 마음 보기` 버튼을 제거했다.
- 친구 페이지의 고정 최대 가로폭을 제거하고, 친구 코드/요청/검색/목록 카드가 화면 폭에 맞춰 유동적으로 확장되도록 조정했다.
- 좌측 사이드바의 `오늘의 한 줄`을 하드코딩 문구가 아니라 `/api/daily-line` 응답으로 렌더링하도록 연결하고, API 실패 시에만 기존 문구를 fallback으로 보여주도록 했다.
- 페이지 로딩 애니메이션을 공통 컴포넌트로 분리하고 전체 배경을 더 어둡게 조정했다.
- `apiFetch` 요청이 진행 중일 때 전역 로딩 오버레이가 뜨도록 연결해, 데이터 조회나 마음 보내기 AI 검사/발송 처리 중에도 편지 봉투 애니메이션과 대기 문구가 표시되도록 했다.
- 로딩 오버레이 아래에 표시할 대기 문구 10개를 추가하고 일정 간격으로 순환되도록 했다.
- 받은 마음 상세 화면에서 수신자 기준 메타 정보는 `도착 시간`만 보이도록 조정했다.
- 받은 마음 상세 화면 상단에 자동 봉투 표지(`thumbnail -> coverImageUrl -> theme 기본 봉투`)를 오버레이 배경처럼 깔고, 본문과 액션 영역은 그 위에서 읽히도록 레이어를 정리했다.
- 로딩 애니메이션을 네모 봉투 본체, 삼각형 플랩, 별 모양 밀랍, 반짝이가 순서대로 붙는 조립형 애니메이션으로 다시 구성했다.
- 메인 화면의 `전체 보기` 버튼에 호버/활성 상태 전환을 추가해 배경, 테두리, 글자색, 그림자, 화살표 위치가 자연스럽게 변하도록 했다.
- `오늘의 한 줄` API 응답 형식을 `{ dailyLine: { date, text, poemTitle, poet } }`에 맞추고, 사이드바에는 본문과 `/ 시 제목, 시인`, `/ 날짜`가 순서대로 표시되도록 조정했다.
